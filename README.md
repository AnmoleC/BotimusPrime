# BotimusPrime
Discord Bot with the following features: <br>
 - Reminders<br>
 - Custom Image responses<br>
 - PSO2 Calender Event Integration<br>
 - Discord Games<br>

Needs 2 command line arguments<br>
- 1st: Bot Token<br>
- 2nd: Google Calender API Oauth token<br>
